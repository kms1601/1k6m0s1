## 자주 사용하는 메타 문자

| 표현식                          | 의미                 |
|------------------------------|--------------------|
| (xy)                         | 문자를 하나로 묶음         |
| x\|y                         | x, y 둘 중 하나        |
| .                            | 아무 문자(와일드 카드)      |
| x?                           | x가 0번 또는 1번        |
| x+                           | x가 1번 이상 반복        |
| x*                           | x가 0번 이상 반복        |
| x\{n\}                       | x가 n번 반복           |
| x\{n,\}                      | x가 n번 이상 반복        |
| x\{,m\}                      | x가 m번 이하 반복        |
| x\{n,m\}                     | x가 n번 이상 m번 이하 반복  |
| [xy]                         | x, y중 하나(2개 이상 가능) |
| [^xy]                        | x, y를 제외(2개 이상 가능) |
| [x-y]                        | x와 y사이의 문자         |
| ^x                           | x로 시작              |
| x$                           | x로 종료              |
| \\|문자 그대로 사용할 때 사용(이스케이프 문자) |

---

## 문자 범주

|표현식|의미|
|-|-|
|\\w|영어 + 숫자 + _로 이루어진 단어|
|\\W|영어 + 숫자 + _로 이루어지지 않은 단어|
|\\b|문자 사이의 공백|
|\\B|문자 사이의 공백이 아닌 것|
|\\d|숫자|
|\\D|숫자가 아닌 것|
|\\s|공백|
|\\S|공백이 아닌 것|
|\\t|탭|

## Flag

정규표현식 맨 뒤에 추가하여 지정한다.

|옵션|의미|
|-|-|
|i|대소문자를 구분하지 않음|
|m|문자열이 여러줄이여도 전부 확인|
|g|문자열 내 모든 패턴을 찾음|
